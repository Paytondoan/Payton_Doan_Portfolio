How to define a table?

A table can be defined using the tag "table"

Describe when to use the follow html elemets: tr, th, thead, tfoot and caption

The "tr" element is used when to indicate you want to make a row of data. Within the tr tag, you use the "th" tag to indicate what you want to put in that row or specific box. "Thead" is used to group all of your header content or "th" tags together in table. "Tfoot" is used when group footer content in a table. Finally, the caption element is the title for the table.

What is the difference between a colspan and rowspan?

A colspan is used if you want a table cell to cover across multiple columns wheras rowspan allows a table cell to go across multiple rows vertically.

What is the difference between cellspacing and cellpadding?

Cellpadding refers to how much space there is between the text and the borders of a specific table cell whereas cellspacing refers to the space between indivduals table cells.

How do you define table borders using css?

You define table borders in css by typing out "table" with a curly bracket and enter in border with a colon then it'll give you options to either change the color or radius.

How do you control border widths using css?

You can control border widths by identifying what you want to change the border of first (ex: table) then within the curly bracket type in "border-width". You can control how thick or thin you want the border by using a number and with "px" after.
